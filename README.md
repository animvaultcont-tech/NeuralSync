# NeuralSync — AI Startup Landing Page

A professional, corporate-style static landing page for a fictional AI/ML startup. Built as a freelance portfolio demo — showcasing clean design, interactive features, and production-ready code.

---

## 🚀 Features

- **Preloader** — Animated branded loader with progress bar
- **Sticky Navbar** — Hide on scroll down, reveal on scroll up, blur on scroll
- **Hero Section** — Live dashboard mockup with animated stats and bars
- **Features Grid** — 6 feature cards including neural net animation and MLOps collab visual
- **How It Works** — 4-step process section
- **Stats Band** — Animated number counters on scroll
- **Pricing Plans** — 3 tiers (Free / Pro / Enterprise) with Monthly/Annual toggle animation
- **Team Section** — 4 team members with LinkedIn/Twitter links
- **Testimonials** — 3 customer quotes grid
- **Newsletter Signup** — Validated email form with success feedback
- **CTA Banner** — Final conversion section
- **Scroll Progress Bar** — Thin blue bar at top tracks page scroll
- **Fully Responsive** — Mobile, tablet, and desktop

## 🛠️ How to Run Locally

1. Unzip the folder
2. Open `index.html` in any modern browser
3. Or use **VS Code Live Server** extension for best results

> ✅ Zero dependencies — no npm, no build step, no frameworks

---

## 🌐 Deploy on GitHub Pages

1. Create a new GitHub repository
2. Upload all files (preserve folder structure)
3. Go to **Settings → Pages → Source: main branch, / (root)**
4. Live at: `https://yourusername.github.io/startup-website`

> Remember to update `robots.txt` and `sitemap.xml` with your actual GitHub Pages URL!
